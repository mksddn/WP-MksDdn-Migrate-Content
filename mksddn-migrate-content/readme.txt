Export and import single pages (and more) with metadata and media.

== Description ==

MksDdn Migrate Content helps to export/import content between WordPress sites.
Initial focus is on single Pages (with meta and ACF fields) and optional media. Future releases may include additional post types and taxonomies.

== Installation ==

1. Upload the `mksddn-migrate-content` folder to the `/wp-content/plugins/` directory, or install via the Plugins page in WordPress.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Tools â†’ Export & Import (plugin page) to export/import content.

== Frequently Asked Questions ==

= Does it support ACF? =
Yes. When ACF is active, field values are exported and imported.

= Does it support media? =
Yes, optionally.
