Complete WordPress site migration plugin. Export or import your database, media, plugins, and themes with ease.

== Description ==

**MksDdn Migrate Content** - Professional WordPress migration and backup solution.

Export your entire WordPress site including database, media files, plugins, and themes to a single archive file. Import the archive to restore or migrate your site to a new location.

**Features:**

* Export entire WordPress site to a single archive file
* Import archive files to restore or migrate sites
* Export database, media, plugins, and themes
* Automatic URL replacement during migration
* Support for large sites
* Secure and reliable

**How It Works:**

1. Install the plugin on your source and destination sites
2. Export your site to a .mksddn file
3. Import using drag-and-drop on your destination site

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mksddn-migrate-content` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'Migrate Content' in the admin menu

== Frequently Asked Questions ==

= What file format does the plugin use? =

The plugin uses .mksddn format for archive files.

= Can I migrate between different hosting providers? =

Yes, the plugin works across all hosting providers.

= Does the plugin support multisite? =

Multisite support is planned for future versions.
